## 🛠️ Tech Stack
This modern portfolio is built using:
- **React.js** & **Vite** (for blazing fast frontend tooling)
- **TypeScript** (for robust type safety)
- **Three.js** & **React Three Fiber (R3F)** (for the 3D Avatar and interactive elements)
- **GSAP** (for buttery smooth scroll animations and reveals)
- **Vanilla CSS** (for precise, lightweight styling)

## ⚙️ Running Locally
If you want to view the code running on your own machine:
1. Clone the repository: `git clone https://github.com/viRAJ357/AUTONOMOUSPORTFOLIO.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Visit `http://localhost:5173` in your browser.
